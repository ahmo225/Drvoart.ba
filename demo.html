🚀 Getting Started

  1. Open index.html in a modern web browser
  2. For best performance, use a local server: python -m http.server 8000
  3. Add your CNC woodworking video as assets/cnc-woodworking.mp4
  4. Customize colors, content, and contact information as needed

Note: The website works perfectly without the video using a beautiful fallback background.
